## 核心结论

1. Python 基础 Q 矩阵当前包含 26 道开放式题目，覆盖 5 个核心 topic：变量与赋值、循环、函数、递归、作用域。
2. 每道题同时标注：Bloom 目标（L1–L6）、5D MIRT 载荷 `a_specialized`、IRT 三参数（难度/区分度/猜测）、关联 misconception（M1–M8）、推荐干预类型。
3. 5D 主导维度按认知类型分配：变量 → K、循环/递归 → S、函数/作用域 → P。
4. 选题策略分三层：warm-up 覆盖性选题、adaptive 自适应选题、probe 探针选题。
5. 当前已知缺口：functions/recursion/scope 缺少 L4 Analyze 层题目；`a_specialized` 仍为 topic 级共享，未来需细化到题目级。
