# 一、项目真正想解决什么问题

很多教育系统其实都在做：

```text
课程
 ↓
学习
 ↓
考试
 ↓
分数
```

而这份研究认为：

```text
分数 ≠ 认知状态
```

例如：

同样考80分的人：

A：

* 概念理解很好
* 策略差
* 容易粗心

B：

* 死记硬背
* 不会迁移

C：

* 会做题
* 不会解释

结果：

```text
分数一样
认知状态完全不同
```

因此作者提出：

认知状态应该是：

```text
S_t = {K,P,M,G,A,E,W,X,U}
```

即：

| 维度 | 含义      |
| -- | ------- |
| K  | 知识      |
| P  | 技能      |
| M  | 元认知     |
| G  | 目标      |
| A  | 注意力     |
| E  | 情绪      |
| W  | 世界模型    |
| X  | 外部Agent |
| U  | 不确定性    |

这实际上是在做：

> 「人的认知操作系统建模」

而不是做教育。

# 二、项目目标是什么

最终目标其实是：

```text
输入：
当前状态 A

输出：

A → B
```

即：

```text
诊断
↓
建模
↓
规划
↓
干预
↓
验证
```

形成闭环：

```text
Observe
→ Diagnose
→ Plan
→ Intervene
→ Re-evaluate
```

和现代 Agent 非常相似：

```text
Observe
Think
Act
Reflect
```

只是对象从 Agent 变成了人。
