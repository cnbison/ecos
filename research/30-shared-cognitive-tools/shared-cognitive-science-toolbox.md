# 认知科学底层工具箱：A→B 与 SGE 的共享基础

分析者：Bisen & Claude

日期：2026-06-13

---

# 背景

在分析 A→B 调研报告与 SelfLab（SGE）的关联性时，得出结论："两者共享认知科学的底层工具箱"。本文具体说明这个工具箱包含什么，以及两个研究如何分别使用这些工具。

---

# 工具 1：经典认知架构

**是什么**：ACT-R、Soar、CLARION、LIDA 等认知架构提供了"心智由哪些模块组成"的理论框架。

**共享的部分**：

| 模块 | A→B 怎么用 | SGE 怎么用 |
|------|-----------|-----------|
| 记忆系统（工作记忆 / 情节记忆 / 语义记忆） | 定义 K（知识）和 P（技能）的存储位置 | SGE Memory Layer 的三层设计直接受此启发 |
| 目标/动机模块 | 定义 G（目标/价值/意图）维度 | SGE Value Layer 的前身 |
| 元认知模块 | 定义 M（元认知/策略）维度 | SGE Reflection Layer 的理论依据 |
| 注意/意识机制 | 定义 A（注意/工作记忆）维度 | LIDA 的全局广播机制启发了 SGE 的"认知失调触发反思"设计 |
| 动作选择机制 | 选择下一步干预 | SGE 中 AI 婴儿在事件中的行为选择 |

**关键区别**：A→B 只使用这些模块作为"状态变量的分类依据"，SGE 则试图让这些模块在 AI 体内真实运行，产生涌现行为。

---

# 工具 2：贝叶斯状态更新

**是什么**：用贝叶斯定理从观测数据推断隐藏状态——先验（prior）× 似然（likelihood）→ 后验（posterior）。

**数学形式**：

```
P(状态 | 观测) ∝ P(观测 | 状态) × P(状态)
```

**共享的部分**：

- **A→B**：用贝叶斯更新追踪知识掌握度。例如，学生做对一道题，K 维度的掌握概率从 0.3 更新到 0.6。
- **SGE**：用贝叶斯更新追踪价值观变化。例如，AI 在事件中选择了"冒险"而非"安全"，价值观向量中"探索"的权重上升。

**底层同构**：两者都是"从行为证据推断隐藏状态"。区别在于：
- A→B 的隐藏状态是离散的（知识点：掌握/未掌握）
- SGE 的隐藏状态是连续的（价值观向量空间中的一个点）

**可迁移性**：A→B 中成熟的状态估计方法（BKT、DKT、IRT）虽然不能直接套用到 SGE，但其"带置信度的状态更新"范式可以作为 SGE Value Layer 量化的工程参考。

# 工具 3：预测加工理论（Predictive Processing）

**是什么**：认知系统的核心功能是不断生成预测，然后用实际观测来修正预测误差。大脑本质上是一个"预测机器"。

**核心公式**：

```
预测误差 = 实际观测 - 预测
状态更新 = f(预测误差)
```

**共享的部分**：

- **A→B**：Identity Layer 预测"作为一个这样的人，我在下一个事件中应该作何反应"，实际事件与预测的偏差驱动状态更新。A→B 的状态转移公式 `S_{t+1} = update(S_t, observation_t, ...)` 本质上就是预测误差驱动的状态更新。
- **SGE**：SGE v0.2 明确引入了预测加工理论——Identity Layer 基于 Narrative 预测自身行为，预测误差是人格修正和情绪涌现的根本源泉。

**关键区别**：
- A→B 的预测误差修正的是"能力状态"（我比预期更会/更不会）
- SGE 的预测误差修正的是"人格状态"（我比预期更勇敢/更怯懦）

**这是最深层的共享**：两个研究的底层数学结构完全同构——都是一个贝叶斯 agent 在环境中通过预测误差不断更新自身状态。区别仅在于"状态"的语义不同。

# 工具 4：双系统认知理论（Dual-Process Theory）

**是什么**：认知分为两个系统——系统 1（快速、直觉、自动）和系统 2（缓慢、反思、费力）。

**共享的部分**：

- **A→B**：大部分日常学习行为（做题、练习）由系统 1 处理，只有遇到困难或错误时才激活系统 2（深度反思、策略调整）。A→B 的"干预选择"本质上是在问：什么时候需要激活系统 2？
- **SGE**：SGE v0.2 明确设计了双系统——系统 1（直觉层，基于现有价值观快速响应）和系统 2（反思层，认知失调时启动深度反思）。

**关键区别**：
- A→B 中的系统 2 是"人主动反思"
- SGE 中的系统 2 是"AI 系统自主触发反思"

**可迁移性**：SGE 的"认知失调触发器"设计可以参考 A→B 中关于"何时需要深度干预"的判断逻辑。

---

# 工具 5：记忆系统分层

**是什么**：认知科学将记忆分为多个系统——工作记忆（短期、容量有限）、情节记忆（事件经历）、语义记忆（抽象知识）、程序记忆（技能和习惯）。

**共享的部分**：

- **A→B**：直接使用这一分层来定义状态向量的维度——K 对应语义记忆，P 对应程序记忆，M 和 A 依赖工作记忆。
- **SGE**：SGE v0.2 的 Memory Layer 设计了三层——工作记忆（当前上下文）、情节记忆（带情感标签的关键事件）、语义记忆（自我认知图谱），直接受 ACT-R 和 Soar 的记忆分层启发。

**关键区别**：
- A→B 的记忆分层是为了"诊断"——判断人在哪个记忆层有缺失
- SGE 的记忆分层是为了"涌现"——让 AI 通过记忆积累形成人格

---

# 工具 6：BDI 模型（信念-欲望-意图）

**是什么**：Agent 的行为可以用三个要素建模——信念（Belief，对世界的认知）、欲望（Desire，想要达成的目标）、意图（Intention，当前承诺的行动计划）。

**共享的部分**：

- **A→B**：用 BDI 来定义 G（目标/价值/意图）和 W（世界模型/信念）维度。
- **SGE**：SGE 的 Identity Layer 本质上是 BDI 的扩展——信念（W = 世界模型）、欲望（G = 价值观）、意图（当前承诺的行为模式），再加上叙事（Narrative）将其串联为连贯的人生故事。

**关键区别**：
- A→B 使用的是经典 BDI，关注"目标驱动的行为选择"
- SGE 在 BDI 之上加了"叙事连续性"——不仅问"你现在相信什么、想要什么"，还问"你为什么变成这样的人"

---

# 工具 7：元认知（Metacognition）

**是什么**：对自己认知过程的认知——知道自己知道什么、不知道什么，能监控自己的策略是否有效，能调节自己的学习方式。

**共享的部分**：

- **A→B**：M 维度直接对应元认知。观测证据包括反思日志、提示使用模式、错误修正行为。
- **SGE**：Reflection Layer 的核心功能就是元认知——AI 对自己的经历进行反思，提取教训，修正价值观。SGE v0.4 规划的 Meta-Cognition Layer 更进一步——AI 开始意识到"自己在反思"。

**关键区别**：
- A→B 的元认知是"观测指标"——通过元认知行为来判断人的学习状态
- SGE 的元认知是"生成机制"——通过元认知来驱动人格的涌现和演化

# 总结

两者共享的不是某个具体算法或技术，而是一整套**理解心智如何运作的概念框架和数学工具**：

- 认知架构告诉我们"心智由什么组成"
- 贝叶斯更新告诉我们"状态如何从证据中推断"
- 预测加工告诉我们"状态为什么以及如何改变"
- 双系统理论告诉我们"什么时候用多少计算资源"
- 记忆分层告诉我们"信息如何组织"
- BDI 告诉我们"行为如何从信念和目标中产生"
- 元认知告诉我们"系统如何自我审视"

A→B 把这些工具用于"教人学会"，SGE 把这些工具用于"让 AI 成为自己"。底层工具箱相同，应用方向不同。
