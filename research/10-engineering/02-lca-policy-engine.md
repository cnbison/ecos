- 主动设计干预实验
- 探索最优策略
- 持续优化 policy

## 干预空间（Bloom → 策略）

| Bloom 层级 | 干预策略 |
|----------|---------|
| Remember | 闪卡 / 间隔复习 / 重复 |
| Understand | 类比 / 可视化 / 讲解 |
| Apply | 变式训练 / 刻意练习 / 步骤练习 |
| Analyze | 拆题 / 比较 / 思维导图 |
| Evaluate | 辩论 / 评判 / 多解比较 |
| Create | 项目学习 / 探究学习 / 开放任务 |
